My Approach

The UX Design Process

Understanding the Challenge

The Core Problem

1

Poor Information Architecture

Navigation was inconsistent across pages, with the bottom nav bar changing between screens. This made it difficult for users to build muscle memory and find features quickly.

2

Confusing Language and Copy

Use of Amharic script mixed with English on the main screen alienated non-native users and appeared like system errors. Microcopy like "PLEASE INSTALL" in the main dashboard was out of place and confusing.

3

Cluttered Dashboard

The "QUICK ACCESS" grid was overwhelming with 8 options, and the secondary grid below added to visual noise without clear categorization, making it difficult for users to find what they needed.

The Solution

Design & Implementation

Original Dashboard (Before)
  • Cluttered layout with 16+ options
  • Unclear iconography
  • Mixed languages creating confusion
  • No financial insights or personalization
Redesigned Dashboard (After)
  • Clean, prioritized layout with 4 main actions
  • Intuitive iconography with labels
  • Consistent language throughout
  • Financial health widget for insights
